# Pecas de reposicao do prototipo

Esta pasta documenta pecas do proprio prototipo que foram substituidas por
versoes impressas com PET reciclado.

Durante o uso do equipamento, algumas pecas impressas inicialmente com materiais
convencionais, especialmente PLA, apresentaram desgaste prematuro ou quebra.
Nesses casos, o filamento PET reciclado produzido pelo projeto foi usado para
fabricar novas versoes das pecas.

## Exemplo documentado

| Peca | Motivo da substituicao | Material usado na reposicao |
| --- | --- | --- |
| Engrenagem do motor NEMA 17 | Quebra ou desgaste durante o funcionamento do prototipo | PET reciclado produzido no projeto |

## Relevancia

Essa reposicao demonstra uma aplicacao direta do material produzido: o filamento
reciclado nao foi usado apenas em corpos de prova, mas tambem em uma peca
funcional do proprio equipamento.
